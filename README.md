# Pex SDK for Python

Python bindings for the [Pex SDK](https://docs.search.pex.com).

### Installation

You can install the Python language bindings like this:


    python3 -m venv env
    . env/bin/activate
    pip install git+https://github.com/Pexeso/pex-sdk-py.gitv4.1.2
